# Myproject-
Напишите регулярное выражение, с помощью которого из URL можно извлечь его домен. Для проверки вашего регулярного выражения будет использован следующий файл с URL-адресами.


Задача — написать скрипт, который будет обрабатывать данный файл в формате json. Скрипт должен сгруппировать покупки по магазинам, в которых они были сделаны. В результате должен получиться массив, состоящий из трех элементов. Внутри каждого элемента содержатся название магазина и все покупки, совершенные в нем. Задачу необходимо решить на Python.


Дан массив из n целых чисел. Нужно выбрать n-1 чисел так, чтобы их произведение было максимальным среди всех возможных n-1 наборов.


Есть два сортированных списка (массива). Нужно написать функцию, которая делает новый сортированный список с пересечением элементов этих двух списков.
Пример:
1-ый список: 1, 2, 2, 5, 7, 14
2-й список: 4, 6, 6, 7, 9, 14, 15
Пересечение: 7, 14


Есть две таблицы:
orders (order_id, promocode_id) - заказы
promocodes (promocode_id, name, discount) - промокоды
Вопросы:
Какая доля заказов с промокодами?
Какой самый популярный промокод (название) и число его использований?
